### Development

Install JS packages

```shell
make install-js
```

Install Python packages

```shell
make install-python
```

Or use common install

```shell
make install
```

Build JS

```shell
make build-js
```

Run Python server

```shell
make run-dev
```

Run JS server for development without Python server.

```shell
make run-js-dev
```

Run tests

```shell
make tests
```
